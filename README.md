# Wishlist API

Сервер для работы с вишлистами к событиям и праздникам.

Пользователь может:
- зарегистрироваться и войти по email и паролю;
- создавать свои вишлисты с датой события;
- добавлять и редактировать позиции подарков;
- делиться публичной ссылкой на вишлист;
- принимать бронирование подарков без авторизации по публичному токену.

## Стек

- Go + chi
- PostgreSQL
- миграции через golang-migrate
- JWT-авторизация

## Запуск

Запуск через Docker Compose: `docker-compose up --build`.
После старта сервер доступен на порту из `HTTP_PORT`.

## Тесты

Запускать из корня проекта:

`go test -race ./internal/usecase/... -cover`

## API

- OpenAPI: `docs/openapi.yaml`

## Конфигурация

Сервис читает параметры из переменных окружения.
Пример значений — в `.env.example`.
